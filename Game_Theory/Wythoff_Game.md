# 4. Wythoff's Game

Two piles. Take from one pile, or the *same* amount from both piles.
- **Cold Positions (Losing States):** Generated by the Golden Ratio $\phi = \frac{1 + \sqrt{5}}{2} \approx 1.618$.
  - $\phi^2 = \phi + 1$, which heavily simplifies computations: $x = \lfloor (y-x)\phi \rfloor$.
  - **Trick:** Always use `long double` for $\phi$ to avoid precision issues on massive constraints ($10^{18}$).

```cpp
#include "../core.h"

const long double GOLDEN_RATIO = (1.0 + sqrt(5.0)) / 2.0;

// O ( 1 )
pair<int, int> get_kth_wythoff_pair(int k) { return {int(k * GOLDEN_RATIO), int(k * GOLDEN_RATIO) + k}; }

// O ( 1 )
bool is_wythoff_losing(int a, int b) {
    if (a > b) swap(a, b);
    // Since phi^2 = phi + 1, we don't need to compute b explicitly with phi^2
    return a == int((b - a) * GOLDEN_RATIO);
}

// O ( N )
vector<pair<int, int>> wythoff_pairs(int n) {
    vector<pair<int, int>> result;
    for (int k = 0; k < n; ++k) result.push_back(get_kth_wythoff_pair(k));
    return result;
}

// O ( 1 )
bool is_valid_wythoff_move(int a, int b, int x, int y) {
    if (x > a || y > b) return false;
    int da = a - x, db = b - y;
    return (da == 0 && db > 0) || (db == 0 && da > 0) || (da == db && da > 0);
}
```
